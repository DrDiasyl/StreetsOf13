cyborg_omnitoolbox
/obj/item/cyborg_omnitoolbox

CYBORG OMNITOOLS Onmi Toolboxs act as a cache of tools for a particular borg's omnitools. Not all borg get a toolbox (as not all borgs use omnitools), and those that do can only have one toolbox. The toolbox keeps track of a borg's omnitool arms, and handles speed upgrades.

Omnitools are the actual tool arms for the cyborg to interact with. When attack_self is called, they can select a tool from the toolbox. The tool is not moved, and instead only referenced in place of the omnitool's own attacks. The omnitool also takes on the tool's sprite, which completes the illusion. In this way, multiple tools are shared between multiple omnitool arms. A multitool's buffer, for example, will not depend on which omnitool arm was used to set it.
